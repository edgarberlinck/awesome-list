## Como Contribuir

Para adicionar um novo recurso a esta categoria:

1. Crie um arquivo `.yml` na pasta `comunidades/`
2. Siga o formato esperado com todos os campos obrigatórios
3. Abra um Pull Request
4. Aguarde revisão
