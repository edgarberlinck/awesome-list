# Patterns of Enterprise Application Architecture

**Author:** Martin Fowler
**Year:** 2002
**Language:** English
**Difficulty:** Advanced
**Topics:** enterprise, architecture, patterns, software-design

## Links

- [Acessar](https://martinfowler.com/books/eaa.html)

## Descrição

Catálogo de padrões amplamente utilizados no desenvolvimento de aplicações
corporativas. Apresenta soluções recorrentes para problemas relacionados a
persistência, lógica de negócio, transações e organização de sistemas complexos.

Embora muitos exemplos reflitam tecnologias da época, os conceitos continuam
relevantes e influenciam frameworks e arquiteturas modernas.
