# Fundamentals of Data Engineering

**Author:** Joe Reis, Matt Housley
**Year:** 2022
**Language:** English
**Difficulty:** Intermediate
**Topics:** data-engineering, data-platforms, analytics, pipelines

## Links

- [Acessar](https://www.oreilly.com/library/view/fundamentals-of-data/9781098108298/)

## Descrição

Introdução moderna à engenharia de dados, cobrindo todo o ciclo de vida dos
dados dentro de uma organização.

Aborda ingestão, armazenamento, processamento, modelagem, qualidade de dados,
governança e arquitetura de plataformas modernas, oferecendo uma visão ampla
e prática da área.

Recomendado tanto para engenheiros de dados quanto para desenvolvedores e
arquitetos que desejam compreender melhor os desafios de sistemas orientados
a dados.
