# Designing Event-Driven Systems

**Author:** Ben Stopford
**Year:** 2018
**Language:** English
**Difficulty:** Advanced
**Topics:** event-driven-architecture, distributed-systems, streaming, architecture

## Links

- [Acessar](https://www.confluent.io/designing-event-driven-systems/)

## Descrição

Explora os fundamentos da arquitetura orientada a eventos e como utilizá-la
para construir sistemas distribuídos escaláveis e desacoplados.

Aborda conceitos como Event Sourcing, CQRS, Event Streams, integração entre
serviços e modelagem baseada em eventos, apresentando exemplos práticos e
discussões sobre os desafios desse estilo arquitetural.

Leitura recomendada para arquitetos e desenvolvedores que trabalham com
sistemas distribuídos modernos e plataformas de streaming de dados.
