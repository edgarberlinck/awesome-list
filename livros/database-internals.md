## Descrição

Uma exploração aprofundada dos mecanismos internos que fazem bancos de dados
modernos funcionarem.

O livro aborda estruturas de armazenamento, índices, árvores B, LSM Trees,
controle de concorrência, recuperação de falhas, processamento distribuído e
outros conceitos fundamentais presentes em sistemas de banco de dados modernos.

Leitura recomendada para desenvolvedores que desejam compreender o que acontece
além das consultas SQL e tomar decisões arquiteturais mais fundamentadas.
