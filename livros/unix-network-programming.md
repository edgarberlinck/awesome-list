## Descrição

Considerado uma das obras definitivas sobre programação de redes em sistemas
Unix e Unix-like.

O livro explora detalhadamente a API de sockets, comunicação entre processos,
protocolos de rede e técnicas utilizadas para construir aplicações de alto
desempenho que dependem de comunicação distribuída.

Embora exija familiaridade com programação em C e conceitos de sistemas
operacionais, continua sendo uma referência clássica para engenheiros que
desejam compreender redes em um nível mais profundo.
