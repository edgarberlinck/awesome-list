# Clean Architecture

**Author:** Robert C. Martin
**Year:** 2017
**Language:** English
**Difficulty:** Intermediate
**Topics:** architecture, software-engineering, design

## Links

- [Acessar](https://www.oreilly.com/library/view/clean-architecture-a/9780134494272/)

## Descrição

Apresenta princípios arquiteturais para construção de sistemas sustentáveis,
testáveis e independentes de frameworks, bancos de dados e interfaces.

Explora conceitos como separação de responsabilidades, inversão de dependência
e organização de sistemas complexos ao longo do tempo.
